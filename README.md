# Recon Bravo

Sneak past enemies in this tactical stealth game created in C# with unity.

### Execution
The file 'Recon Bravo' compiled for Mac OS X can be found under the 'Release' directory.

### Controls
- W/A/S/D or up/down/left/right to move
